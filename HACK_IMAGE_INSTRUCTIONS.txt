ИНСТРУКЦИЯ ПО НАСТРОЙКЕ ФОНА ДЛЯ EASTER EGG

Для работы кинематографического взлома нужно:

1. Подготовить изображение:
   - Любая картинка (рекомендуется 200x200px или больше)
   - Формат: JPG, PNG, GIF
   - Название файла: любое (например: skull.jpg, logo.png, hack.jpg)

2. Поместить файл изображения в ту же папку где находится cinematic-hack.js

3. Открыть файл cinematic-hack.js и заменить ВСЕ вхождения:
   'your-image.jpg' 
   на название вашего файла, например:
   'skull.jpg'

4. Сохранить файл

ЭФФЕКТ:
- Картинка появится 200+ раз на экране
- Будет вращаться, масштабироваться и накладываться
- Создаст хаотичный пугающий эффект
- Разные цветовые фильтры и эффекты наложения

ТЕКУЩЕЕ СОСТОЯНИЕ:
- Местоположение: "Россия, Ставрополь" ✓
- Определение macOS: исправлено ✓
- Курсор: скрывается во время взлома ✓
- Фон: готов к замене изображения ⚠️

НУЖНО СДЕЛАТЬ:
Заменить 'your-image.jpg' на реальный файл изображения
